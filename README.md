# 📊 EstiMate — универсальный конструктор смет

**EstiMate** — это лёгкое, но мощное веб-приложение для создания иерархических смет, расчёта стоимости и аналитики. Работает полностью в браузере, не требует сервера и сохраняет все данные локально.


## 🚀 Возможности

- **Гибкая структура сметы**  
  Создавайте неограниченную вложенность блоков и статей. Каждый блок может содержать подблоки или позиции расходов.

- **Редактирование в реальном времени**  
  Изменяйте названия, количество, цену, единицы измерения и примечания. Все суммы пересчитываются мгновенно.

- **Готовые пресеты**  
  Начните с шаблона: «Запуск продукта», «Строительство», «Мероприятие», «Рекламная кампания» и др.

- **Сохранение смет**  
  До 4 смет хранятся в `localStorage` браузера. Можно перезаписывать существующие или удалять.

- **Автоматическое восстановление сессии**  
  При возвращении на страницу вы можете продолжить работу с последней редактируемой сметой.

- **Экспорт**  
  - **Excel (.xlsx)** — с помощью библиотеки SheetJS.  
  - **PDF** — через встроенный диалог печати браузера с форматированной таблицей и аналитическим блоком.

- **Аналитика**  
  Встроенная панель с круговой диаграммой распределения затрат по блокам и сводными показателями (общая сумма, количество статей, средняя стоимость, крупнейший блок).

- **Тёмная и светлая темы**  
  Переключайтесь одним кликом, настройка сохраняется.

- **Адаптивный интерфейс**  
  Удобно работать как на десктопе, так и на планшете.

## 🛠️ Технологии

- Чистый **HTML5**, **CSS3**, **JavaScript (ES6)**
- [SortableJS](https://sortablejs.github.io/Sortable/) — drag & drop списков
- [Feather Icons](https://feathericons.com/) — иконки интерфейса
- [SheetJS (xlsx)](https://sheetjs.com/) — генерация Excel-файлов
- [Chart.js 4](https://www.chartjs.org/) — визуализация аналитики
- [Google Fonts (Inter)](https://fonts.google.com/specimen/Inter) — современный шрифт

## 📂 Структура проекта

Весь код находится в одном файле `EstiMate.html`. Никаких дополнительных зависимостей или сборки не требуется.
